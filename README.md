# Five Nights at Freddy's Skin Pack – Minecraft Legacy Console Edition

A Skin Pack for **Minecraft Legacy Console Edition** that adds characters from *Five Nights at Freddy's* with custom geometry for a more authentic look.

## Installation

1. Make sure you have **Minecraft Legacy Console Edition** installed.
2. Download the skin pack files from this repository.
3. Copy the files from the folder to:
   `C:\Users\<YourUsername>\LCEWindows64\Windows64Media\DLC`
4. Open Minecraft, navigate to the skin packs section, and equip your favorite FNaF character!
